# Hotel-Vaibhava-Grand
Hotel Vaibhava Grand is a modern and welcoming hotel in Chitradurga, Karnataka. Established in 2023, we offer clean, comfortable AC &amp; Non-AC rooms, multi-cuisine dining, and a spacious banquet hall for weddings and events (up to 300 guests). Located near Chitradurga Fort and the Railway Bridge, we provide excellent hospitality at affordable prices.
